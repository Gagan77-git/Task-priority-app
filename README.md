📌 Task Priority Web App

A web-based task management application that allows users to manage their daily tasks efficiently with priority levels and secure user authentication.

🚀 Features
🔐 User Authentication (Signup & Login using Supabase)
📝 Add, Update, Delete tasks
✅ Mark tasks as completed
📊 Sort tasks by priority (High, Medium, Low)
👤 User-specific data (each user sees only their tasks)
🔒 Secure database using Row Level Security (RLS)
🎨 Simple and responsive UI

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Supabase (Database + Authentication)
Hosting: GitHub Pages / Netlify

⚙️ How It Works
Users sign up and log in using email and password
Each task is stored in the database with a unique user_id
Row Level Security ensures users can only access their own tasks
Tasks can be created, updated, marked as done, or deleted
